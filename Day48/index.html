🔍 How this was researched ›

This comparison combines two kinds of numbers: (1) directly sourced facts — TIOBE percentages, Stack Overflow survey figures, GitHub Octoverse contributor counts, and Glassdoor/Indeed/AmbitionBox salary data — quoted or closely paraphrased from the cited pages, and (2) 1–10 "synthesized" ratings, which are this tool's own judgment calls that translate those raw facts into a comparable scale. The synthesized ratings are clearly labeled in each language's detail view and are not themselves sourced numbers — they're an analyst's read of the sourced facts sitting next to them.

Conflicts between sources

"Most popular" means different things. TIOBE (search-engine mentions), Stack Overflow (self-reported usage), and GitHub Octoverse (contributor/repo counts) all rank languages differently. JavaScript leads Stack Overflow's usage survey (66%) and Octoverse's new-repo count, while Python leads TIOBE. This tool reports all three rather than picking one "winner" metric.
Rust salary data has a small sample size. Glassdoor's India figure for Rust developers (₹12L average) is based on far fewer submitted salaries than Python or Node.js listings (dozens vs. hundreds), so it's noisier and skews toward senior/specialist roles. This is flagged directly in Rust's job-market detail.
Go's TIOBE ranking dropped from #7 to #13 between July 2025 and July 2026, which could misleadingly suggest declining relevance. Industry commentary (cited) notes this reflects search/discussion volume, not job-market abandonment — Go remains a standard choice for cloud infrastructure and APIs.

What wasn't independently verified

Salary figures for India are drawn from aggregator sites (Glassdoor, AmbitionBox via secondary write-ups, Indeed) rather than government wage statistics, and reflect self-reported submissions. Package-registry counts (PyPI, crates.io) are point-in-time snapshots from the cited dates and will drift over time. No numbers in this tool were invented — every score in a detail panel traces to a citation in the Sources panel below.
